# Salary Prediction System

A Machine Learning project that predicts employee salary based on:

- Age
- Experience
- Gender
- Education
- Location
- Job Title

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

## Model

- Multiple Linear Regression

## Performance

- R² Score: 0.87

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app/app.py
```
